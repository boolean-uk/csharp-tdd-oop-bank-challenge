## User Stories

```
As a customer,
So I can safely store use my money,
I want to create a current account.

| Class | Properties | Methods         | Scenario                                | Outputs                        |
|-------|------------|-----------------|-----------------------------------------|--------------------------------|
| Core  |            | createUserAcc() | Create account with username & password | Account stored in Account list |
|       |            | creatCurrentAcc()|	


As a customer,
So I can save for a rainy day,
I want to create a savings account.

| Class | Properties | Methods           | Scenario                                      | Outputs                       |
|-------|------------|-------------------|-----------------------------------------------|-------------------------------|
| Core  |            | createSavingAcc() | Create a saving account with transaction list | Saving account stored in User |



As a customer,
So I can keep a record of my finances,
I want to generate bank statements with transaction dates, amounts, and balance at the time of transaction.

| Class | Properties | Methods         | Scenario             | Outputs                   |
|-------|------------|-----------------|----------------------|---------------------------|
| Core  |            | BankStatement() | Bank data collecting | generated bank statements |



As a customer,
So I can use my account,
I want to deposit and withdraw funds.

| Class | Properties | Methods                          | Scenario                                          | Outputs         |
|-------|------------|----------------------------------|---------------------------------------------------|-----------------|
| Core  |            | DepositAmount() WithdrawAmount() | Ableity to deposit and withdraw money by Customer | Transaction log |

```
